# Notes App (Vanilla TypeScript SPA)

A simple notes application built using **TypeScript**, **DOM APIs**, and **History API** (no frameworks).

## Features
- Create notes
- View all notes
- View individual note details
- Client-side routing
- Data stored in browser (localStorage)

## Routes
- `/` – List all notes
- `/create` – Create a new note
- `/note/:id` – View note details

## Tech Stack
- TypeScript
- HTML
- Tailwind CSS
- Browser History API

## Run Locally
```bash
npm install
npm run dev
```
